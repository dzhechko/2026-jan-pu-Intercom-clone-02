# Руководство оператора КоммуниК

## Обзор

Оператор -- сотрудник службы поддержки, работающий с клиентскими диалогами. КоммуниК автоматически обнаруживает потенциальных покупателей (PQL) в диалогах и предоставляет CRM-контекст для более эффективной работы.

## Вход в систему

1. Откройте рабочее пространство по адресу: `https://yourdomain.ru/workspace`
2. Введите email и пароль, выданные администратором
3. После входа вы автоматически подключаетесь к WebSocket для получения обновлений в реальном времени

## Рабочее пространство оператора

Интерфейс состоит из трех панелей:

```
+-------------------+------------------------+-------------------+
|                   |                        |                   |
|  Список диалогов  |     Окно чата          |  Правая панель    |
|                   |                        |                   |
|  - Фильтры       |  - История сообщений   |  - Информация     |
|  - Поиск         |  - Поле ввода          |    о клиенте      |
|  - PQL бейджи    |  - Быстрые ответы      |  - Memory AI      |
|  - Статусы       |  - Вложения            |  - PQL детали     |
|                   |                        |  - CRM контекст   |
+-------------------+------------------------+-------------------+
```

### Левая панель: Список диалогов

- Отображает все активные и назначенные диалоги
- PQL-бейджи показывают тир лида (HOT/WARM/COLD)
- Непрочитанные диалоги выделены жирным шрифтом
- Сортировка: по времени последнего сообщения, по PQL Score, по приоритету

### Центральная панель: Окно чата

- Полная история переписки с клиентом
- Поле ввода с поддержкой форматирования
- Индикатор набора текста клиентом
- Информация о канале (виджет, Telegram, VK Max)

### Правая панель: Контекст

- Информация о клиенте (имя, email, канал)
- Memory AI: данные из CRM (сделки, история покупок, контакт)
- PQL детали: обнаруженные сигналы, Score, тир
- История предыдущих диалогов

## Обработка диалогов

### Назначение диалога

Новые диалоги попадают в общую очередь. Для назначения себе:

1. Кликните на диалог в списке
2. Нажмите кнопку "Взять в работу" или используйте горячую клавишу `Ctrl+Enter`
3. Диалог переместится в раздел "Мои диалоги"

### Ответ клиенту

1. Введите текст в поле ввода
2. Нажмите `Enter` для отправки или `Shift+Enter` для новой строки
3. Для вложений используйте кнопку скрепки или перетащите файл

### Закрытие диалога

1. Нажмите кнопку "Закрыть диалог" или `Ctrl+Shift+X`
2. Выберите причину закрытия (решено, спам, дубликат)
3. Диалог перемещается в архив

## PQL индикаторы

КоммуниК автоматически анализирует каждое сообщение клиента и определяет покупательское намерение.

### Тиры PQL

| Тир | Score | Цвет | Значок | Действие |
|-----|-------|------|--------|----------|
| **HOT** | >= 0.80 | Красный | Огонь | Немедленно обратить внимание, предложить продукт |
| **WARM** | >= 0.65 | Оранжевый | Термометр | Уточнить потребности, предоставить информацию |
| **COLD** | < 0.65 | Серый | -- | Стандартная обработка обращения |

### Как выглядят PQL индикаторы

- В списке диалогов: цветной бейдж рядом с именем клиента
- В окне чата: баннер вверху с PQL Score и обнаруженными сигналами
- В правой панели: детальная разбивка по сигналам

### Обратная связь по PQL

Если система ошиблась в определении PQL:

1. Нажмите на PQL бейдж
2. Выберите "Подтвердить" или "Отклонить"
3. Добавьте комментарий (опционально)

Обратная связь улучшает точность детекции для вашей компании.

## Memory AI

Memory AI автоматически загружает контекст клиента из CRM (amoCRM) перед тем, как вы начнете отвечать.

### Что отображается

- **Контакт:** имя, телефон, email, компания
- **Сделки:** активные сделки, сумма, стадия воронки
- **История:** предыдущие покупки и обращения
- **Заметки:** комментарии менеджеров в CRM

### Как это помогает

Вместо того чтобы переключаться между поддержкой и CRM, вы видите всю информацию о клиенте прямо в рабочем пространстве. Это позволяет:

- Персонализировать ответ ("Вижу, у вас тариф Basic с марта")
- Предложить апгрейд на основе истории
- Не задавать вопросы, ответы на которые уже есть в CRM

## Быстрые ответы

Часто используемые ответы доступны через горячие клавиши:

| Клавиша | Действие |
|---------|----------|
| `Alt+1` | Приветствие |
| `Alt+2` | Запрос уточнения |
| `Alt+3` | Отправка на другой отдел |
| `Alt+4` | Ответ о ценах/тарифах |
| `Alt+5` | Прощание и закрытие |

Быстрые ответы настраиваются администратором для каждого тенанта.

## Горячие клавиши

| Сочетание | Действие |
|-----------|----------|
| `Ctrl+Enter` | Взять диалог в работу |
| `Enter` | Отправить сообщение |
| `Shift+Enter` | Новая строка в сообщении |
| `Ctrl+Shift+X` | Закрыть диалог |
| `Ctrl+K` | Поиск по диалогам |
| `Ctrl+/` | Показать все горячие клавиши |
| `Ctrl+1` | Перейти к списку диалогов |
| `Ctrl+2` | Перейти к окну чата |
| `Ctrl+3` | Перейти к правой панели |
| `Alt+1..5` | Быстрые ответы |
| `Esc` | Закрыть модальное окно |
| `Ctrl+Shift+F` | Развернуть/свернуть фильтры |
| `Ctrl+N` | Перейти к следующему диалогу |
| `Ctrl+P` | Перейти к предыдущему диалогу |

## Фильтрация и поиск

### Фильтры

| Фильтр | Опции |
|--------|-------|
| Статус | Все, Открытые, Мои, Закрытые |
| PQL тир | HOT, WARM, COLD, Все |
| Канал | Виджет, Telegram, VK Max |
| Период | Сегодня, Неделя, Месяц, Произвольный |
| Оператор | Мои, Конкретный оператор, Неназначенные |

### Поиск

- `Ctrl+K` -- открыть строку поиска
- Поиск по имени клиента, email, содержанию сообщений
- Результаты обновляются по мере ввода
- Поддержка кириллицы и латиницы

## Уведомления

### В рабочем пространстве

- Звуковой сигнал при новом сообщении
- Всплывающее уведомление при обнаружении HOT PQL
- Счетчик непрочитанных на вкладке браузера

### Email и Telegram

Настраиваются администратором. Вы можете получать уведомления о:
- Новых назначенных диалогах
- Обнаруженных PQL (по тиру)
- Напоминания о неотвеченных диалогах

## Советы по эффективной работе

1. **Следите за PQL индикаторами** -- HOT лиды в приоритете
2. **Используйте Memory AI** -- не переспрашивайте то, что уже есть в CRM
3. **Давайте обратную связь по PQL** -- это улучшает точность системы
4. **Используйте горячие клавиши** -- экономьте время на рутинных операциях
5. **Фильтруйте по PQL тиру** -- обрабатывайте HOT и WARM лиды в первую очередь
