# Руководство по интерфейсу КоммуниК

## Chat Widget

### Встраивание на сайт

Добавьте следующий код перед закрывающим тегом `</body>`:

```html
<script>
  window.kommunikSettings = {
    tenantId: 'your-tenant-uuid',
    position: 'bottom-right',     // bottom-right | bottom-left
    primaryColor: '#6366f1',      // основной цвет виджета
    title: 'Поддержка',           // заголовок виджета
    subtitle: 'Обычно отвечаем за 5 минут',
    placeholder: 'Напишите сообщение...',
    welcomeMessage: 'Здравствуйте! Чем можем помочь?',
    locale: 'ru'                  // ru | en
  };
</script>
<script src="https://yourdomain.ru/widget/kommunik.js" async></script>
```

### Кастомизация

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `tenantId` | string | -- | ID тенанта (обязательный) |
| `position` | string | `bottom-right` | Позиция кнопки виджета |
| `primaryColor` | string | `#6366f1` | Основной цвет (HEX) |
| `title` | string | `Поддержка` | Заголовок окна чата |
| `subtitle` | string | -- | Подзаголовок |
| `placeholder` | string | `Напишите сообщение...` | Placeholder поля ввода |
| `welcomeMessage` | string | -- | Приветственное сообщение |
| `locale` | string | `ru` | Язык интерфейса |
| `zIndex` | number | `9999` | z-index виджета |

### Программное управление

```javascript
// Открыть виджет
window.Kommunik.open();

// Закрыть виджет
window.Kommunik.close();

// Отправить пользовательские данные
window.Kommunik.identify({
  name: 'Иван Петров',
  email: 'ivan@company.ru',
  plan: 'professional'
});

// Обработка событий
window.Kommunik.on('message:sent', function(data) {
  console.log('Сообщение отправлено:', data);
});

window.Kommunik.on('widget:opened', function() {
  console.log('Виджет открыт');
});
```

## Workspace: Рабочее пространство оператора

### 3-панельная раскладка

```
+--------------------+---------------------------+--------------------+
|                    |                           |                    |
|   LEFT PANEL       |      CENTER PANEL         |   RIGHT PANEL      |
|   (280px)          |      (flex-grow)          |   (320px)          |
|                    |                           |                    |
|   Список           |      Окно чата            |   Контекст         |
|   диалогов          |                           |   клиента          |
|                    |                           |                    |
+--------------------+---------------------------+--------------------+
```

- **Responsive:** На экранах < 768px панели переключаются табами
- **Resizable:** Границы панелей можно перетаскивать мышью
- **Collapsible:** Правая панель сворачивается по `Ctrl+3`

### Компоненты интерфейса

#### Заголовок (Header)

```
+--------------------------------------------------------------+
| KоммуниК  | Поиск (Ctrl+K)  |  Уведомления  |  Профиль     |
+--------------------------------------------------------------+
```

- Логотип и название компании
- Глобальный поиск по диалогам
- Счетчик непрочитанных уведомлений
- Меню профиля (настройки, выход)

#### Список диалогов (Left Panel)

```
+----------------------------+
| Фильтры: Все | Мои | PQL  |
+----------------------------+
| [HOT] Иван Петров    14:32|
| "сколько стоит Pro..."     |
+----------------------------+
| [WARM] Мария Сидорова 14:15|
| "есть пробный период..."  |
+----------------------------+
| Алексей К.           13:58 |
| "не работает кнопка..."   |
+----------------------------+
```

Каждый элемент списка содержит:
- PQL бейдж (HOT/WARM) или пустое место (COLD/нет PQL)
- Имя клиента
- Превью последнего сообщения (обрезается до 40 символов)
- Время последнего сообщения
- Индикатор непрочитанного (синяя точка)
- Канал (иконка: виджет, Telegram, VK)

#### Окно чата (Center Panel)

```
+------------------------------------------+
| Иван Петров | Telegram | [HOT PQL: 0.87] |
+------------------------------------------+
|                                          |
| [14:30] Клиент:                          |
| Здравствуйте! Сколько стоит тариф Pro?   |
|                                          |
| [14:32] Оператор:                        |
| Добрый день! Тариф Pro стоит...          |
|                                          |
+------------------------------------------+
| [Поле ввода]              [Прикрепить] > |

```

## PQL визуализация

### Цвета тиров

| Тир | Фоновый цвет | Цвет текста | CSS-класс |
|-----|--------------|-------------|-----------|
| **HOT** | `#FEE2E2` (red-100) | `#DC2626` (red-600) | `pql-tier-hot` |
| **WARM** | `#FEF3C7` (amber-100) | `#D97706` (amber-600) | `pql-tier-warm` |
| **COLD** | `#F3F4F6` (gray-100) | `#6B7280` (gray-500) | `pql-tier-cold` |

### Бейджи в списке диалогов

```
HOT  → Красный бейдж с иконкой огня и текстом "HOT"
WARM → Оранжевый бейдж с иконкой термометра и текстом "WARM"
COLD → Не отображается (нет бейджа)
```

### PQL баннер в окне чата

При обнаружении PQL в верхней части окна чата появляется баннер:

```
+--------------------------------------------------+
| [Огонь] HOT PQL (Score: 0.87)                    |
| Сигналы: запрос цены, сравнение тарифов          |
| [Подтвердить]  [Отклонить]                       |
+--------------------------------------------------+
```

### PQL детали в правой панели

```
+---------------------------+
| PQL Intelligence          |
+---------------------------+
| Score: 0.87 [HOT]         |
| _________________________ |
| Обнаруженные сигналы:     |

+---------------------------+
```

### Графики

1. **Тренд PQL по дням** -- линейный график, разбивка по тирам (HOT/WARM)
2. **Воронка PQL** -- PQL обнаружен -> Подтвержден -> Сделка создана -> Сделка закрыта
3. **Распределение по каналам** -- круговая диаграмма (Виджет, Telegram, VK Max)
4. **Топ операторов по конверсии** -- горизонтальная столбчатая диаграмма
